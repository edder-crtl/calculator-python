Este proyecto es parte de mi proceso de aprendizaje y consolidación de bases en Python. El objetivo principal es desarrollar una calculadora funcional mientras practico conceptos fundamentales de programación y buenas prácticas de desarrollo. La idea es documentar mi avance de forma transparente y demostrar cómo evolucionan mis habilidades a medida que trabajo en proyectos más completos.

La calculadora incluye operaciones aritméticas esenciales como suma, resta, multiplicación y división. Además, implementa manejo de errores utilizando estructuras try/except para validar entradas del usuario y evitar fallos por datos incorrectos. También incorpora un sistema básico de historial que registra las operaciones realizadas, lo que me permite practicar la gestión de datos dentro del programa.

Este proyecto busca reforzar conocimientos clave como:

Uso de funciones para organizar el código de manera modular.

Control de flujo mediante condicionales y bucles.

Validación de datos y manejo de excepciones.

Interacción con el usuario desde consola.

Estructuración de proyectos y documentación básica.

La calculadora representa un paso inicial dentro de mi portafolio como desarrollador de software. Mi meta es continuar ampliando este proyecto con nuevas características, optimizaciones y aprendizajes, como integración de más operaciones, mejoras en la interfaz o migración futura hacia una versión gráfica o web.

Este repositorio registra mi progreso real en el camino de convertirme en un mejor programador y sirve como evidencia concreta de mi proceso formativo.
